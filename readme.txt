A simple, one-click way for visitors to report broken links directly from your posts.

== Description ==
Kreativ Report Broken Link lets your readers report broken links with one click.
Each report is automatically stored in your dashboard and emailed to you.

== Features ==
* Automatic “Report Broken Link” button at the end of every post
* One-click AJAX reporting — no forms
* Dashboard page with reports list and status management (New, Resolved, Ignored)
* Email notifications to admin or custom address
* Lightweight and GDPR-friendly (no cookies or personal data)

== Installation ==
1. Upload the plugin ZIP via *Plugins → Add New → Upload*.
2. Activate it.
3. Go to *Kreativ Broken Links → Settings* to configure your email.

== Frequently Asked Questions ==
= Does it slow down my site? =
No — it loads a tiny inline JavaScript only on single posts.
